# `x86decomp serve`

## Usage

```text
usage: x86decomp serve [-h] [--host HOST] [--port PORT] project
```

## Arguments

| Argument | Details |
| --- | --- |
| `project` | required · type: `path`. |
| `--host` | default: `'127.0.0.1'`. |
| `--port` | type: `int` · default: `8765`. |
