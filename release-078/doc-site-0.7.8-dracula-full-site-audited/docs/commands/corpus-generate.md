# `x86decomp corpus-generate`

## Usage

```text
usage: x86decomp corpus-generate [-h] [--cases-per-family CASES_PER_FAMILY]
                                 [--seed SEED] [--c-only] [--cpp-only]
                                 [--report REPORT]
                                 output_directory
```

## Arguments

| Argument | Details |
| --- | --- |
| `output_directory` | required · type: `path`. |
| `--cases-per-family` | type: `int` · default: `8`. |
| `--seed` | type: `int` · default: `2262745310`. |
| `--c-only` | nargs: `0` · default: `False`. |
| `--cpp-only` | nargs: `0` · default: `False`. |
| `--report` | type: `path`. |
