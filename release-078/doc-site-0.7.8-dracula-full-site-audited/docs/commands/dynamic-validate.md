# `x86decomp dynamic-validate`

## Usage

```text
usage: x86decomp dynamic-validate [-h] [--target-base TARGET_BASE]
                                  [--candidate-base CANDIDATE_BASE]
                                  [--report REPORT]
                                  target candidate harness
```

## Arguments

| Argument | Details |
| --- | --- |
| `target` | required · type: `path`. |
| `candidate` | required · type: `path`. |
| `harness` | required · type: `path`. |
| `--target-base` | type: `int`. |
| `--candidate-base` | type: `int`. |
| `--report` | type: `path`. |
