# `x86decomp init`

## Usage

```text
usage: x86decomp init [-h] [--reference-binary] binary project
```

## Arguments

| Argument | Details |
| --- | --- |
| `binary` | required · type: `path`. |
| `project` | required · type: `path`. |
| `--reference-binary` | nargs: `0` · default: `False`. |
