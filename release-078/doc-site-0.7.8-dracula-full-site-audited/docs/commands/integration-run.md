# `x86decomp integration-run`

```text
usage: x86decomp integration-run [-h] [--allow-host-execution]

```

| Argument | Details |
| --- | --- |
| `manifest` | required · type: `path`. |
| `--allow-host-execution` | nargs: `0` · default: `False`. |
| `--report` | type: `path`. |
