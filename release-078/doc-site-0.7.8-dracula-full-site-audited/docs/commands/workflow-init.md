# `x86decomp workflow-init`

## Usage

```text
usage: x86decomp workflow-init [-h] [--mode {matching,functional}]
                               project function_id
```

## Arguments

| Argument | Details |
| --- | --- |
| `project` | required · type: `path`. |
| `function_id` | required. |
| `--mode` | choices: `matching`, `functional`. |
