# `x86decomp diff-bytes`

## Usage

```text
usage: x86decomp diff-bytes [-h] [--report REPORT]
                            [--max-mismatches MAX_MISMATCHES]
                            target candidate
```

## Arguments

| Argument | Details |
| --- | --- |
| `target` | required · type: `path`. |
| `candidate` | required · type: `path`. |
| `--report` | type: `path`. |
| `--max-mismatches` | type: `int` · default: `64`. |
