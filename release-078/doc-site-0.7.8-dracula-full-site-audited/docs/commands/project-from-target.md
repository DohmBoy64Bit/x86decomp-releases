# `x86decomp project-from-target`

## Usage

```text
usage: x86decomp project-from-target [-h] [--reference-binary]
                                     target_pack project
```

## Arguments

| Argument | Details |
| --- | --- |
| `target_pack` | required · type: `path`. |
| `project` | required · type: `path`. |
| `--reference-binary` | nargs: `0` · default: `False`. |
