# `x86decomp drcov-run`

## Usage

```text
usage: x86decomp drcov-run [-h] [--drrun DRRUN] [--program-arg PROGRAM_ARG]
                           [--timeout TIMEOUT] [--report REPORT]
                           executable output_directory
```

## Arguments

| Argument | Details |
| --- | --- |
| `executable` | required · type: `path`. |
| `output_directory` | required · type: `path`. |
| `--drrun` | type: `path`. |
| `--program-arg` | default: `[]`. |
| `--timeout` | type: `int` · default: `300`. |
| `--report` | type: `path`. |
