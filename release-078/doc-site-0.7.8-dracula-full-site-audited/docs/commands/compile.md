# `x86decomp compile`

## Usage

```text
usage: x86decomp compile [-h] [--report REPORT] [--extra-arg EXTRA_ARG]
                         [--cache CACHE]
                         profile source output
```

## Arguments

| Argument | Details |
| --- | --- |
| `profile` | required · type: `path`. |
| `source` | required · type: `path`. |
| `output` | required · type: `path`. |
| `--report` | type: `path`. |
| `--extra-arg` | default: `[]`. |
| `--cache` | type: `path`. |
