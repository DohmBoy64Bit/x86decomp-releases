# `x86decomp content-put`

## Usage

```text
usage: x86decomp content-put [-h] [--media-type MEDIA_TYPE]
                             [--reference REFERENCE] [--kind KIND]
                             [--owner OWNER]
                             store file
```

| Argument | Details |

| `store` | required · type: `path`. |
| `file` | required · type: `path`. |
| `--media-type` | default: `'application/octet-stream'`. |
| `--reference` | — |
| `--kind` | default: `'artifact'`. |
