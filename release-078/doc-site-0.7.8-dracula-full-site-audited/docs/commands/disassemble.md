# `x86decomp disassemble`

## Usage

```text
usage: x86decomp disassemble [-h] [--base BASE] [--architecture {x86,x86_64}]
                             [--report REPORT]
                             code
```

## Arguments

| Argument | Details |
| --- | --- |
| `code` | required · type: `path`. |
| `--base` | type: `int` · default: `0`. |
| `--architecture` | choices: `x86`, `x86_64` · default: `'x86'`. |
| `--report` | type: `path`. |
