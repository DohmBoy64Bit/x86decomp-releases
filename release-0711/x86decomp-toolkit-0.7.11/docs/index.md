# x86decomp toolkit 0.7.11

x86decomp is an evidence-governed toolkit for static analysis, reconstruction, and bounded
validation of native x86 PE32 and x86-64 PE32+ artifacts. It does not execute input binaries during
parsing, and optional execution paths require explicit authorization and bounded environments.

Use the navigation to review supported scope, architecture, contracts, command surfaces, security
guardrails, project operations, and release verification. The source-controlled Markdown files are
the documentation source of truth; the MkDocs Material build is a synchronized presentation of
those files.
