x86decomp candidate

Create, compare, evaluate, and transition reconstruction candidates.

currentcanonical group7 runnable paths

Quick example

x86decomp candidate --help
canonicalgovernance

x86decomp candidate add-file

Adds file to the selected candidates.

Usage

x86decomp candidate add-file [-h] candidate_id source relative_path

Example

x86decomp candidate add-file ./candidate.bin ./candidate.c src/candidate.c

Arguments

ArgumentWhat it controls
candidate_id
required		Identifier of the candidate being operated on.
source
required		Path to the source input.
relative_path
required		Path to the relative used by the command.
canonicalgovernance

x86decomp candidate compare

Compares candidates and reports meaningful differences.

Usage

x86decomp candidate compare [-h] left_id right_id

Example

x86decomp candidate compare example-001 example-001

Arguments

ArgumentWhat it controls
left_id
required		Identifier of the first record to compare.
right_id
required		Identifier of the second record to compare.

x86decomp candidate create

Creates a new candidates record.

Usage

x86decomp candidate create [-h] [--campaign-id CAMPAIGN_ID] [--parent PARENT] [--objective-json OBJECTIVE_JSON] branch_name

Example

x86decomp candidate create candidate-main

Arguments

ArgumentWhat it controls
branch_name
required		Controls branch name for this command.
--campaign-idIdentifier of the campaign being operated on.
--parentControls parent for this command.
--objective-jsonPath to the JSON file containing objective data.

x86decomp candidate evaluate

Evaluates the selected candidates against current evidence.

Usage

x86decomp candidate evaluate [-h] [--value VALUE] [--details-json DETAILS_JSON] candidate_id metric status

Example

x86decomp candidate evaluate ./candidate.bin exact-bytes active

Arguments

ArgumentWhat it controls
candidate_id
required		Identifier of the candidate being operated on.

required		Controls metric for this command.
status
required		Requested or reported status.
--valueControls value for this command.

x86decomp candidate transition

Moves the selected candidates through an approved state transition.

Usage

x86decomp candidate transition [-h] --reason REASON candidate_id state

Example

x86decomp candidate transition ./candidate.bin active --reason reviewed-change

Arguments

ArgumentWhat it controls
candidate_id
required		Identifier of the candidate being operated on.
state
required		Requested workflow state.
--reason
required		Controls reason for this command.
