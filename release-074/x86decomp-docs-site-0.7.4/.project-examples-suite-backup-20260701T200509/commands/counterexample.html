x86decomp counterexample add

Adds a record to the selected counterexamples.

Usage

x86decomp counterexample add [-h] --predicate-json PREDICATE_JSON [--provenance-json PROVENANCE_JSON] scope_kind scope_id payload

Example

x86decomp counterexample add function pe-rva:00001000 example --predicate-json matches

Arguments

ArgumentWhat it controls
scope_kind
required		Controls scope kind for this command.
scope_id
required		Identifier of the scope used by the command.
payload
required		Controls payload for this command.
--predicate-json
required		Path to the JSON file containing predicate data.
--provenance-jsonPath to the JSON file containing provenance data.

x86decomp counterexample promote

Promotes the selected counterexamples into durable project data.

Usage

x86decomp counterexample promote [-h] counterexample_id destination

Example

x86decomp counterexample promote 1 example

Arguments

ArgumentWhat it controls
counterexample_id
required		Identifier of the counterexample used by the command.
destination
required		Controls destination for this command.
