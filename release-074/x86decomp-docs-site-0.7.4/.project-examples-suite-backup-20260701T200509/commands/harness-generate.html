x86decomp harness-generate

Harness generate.

Usage

x86decomp harness-generate [-h] [--pointer-parameters POINTER_PARAMETERS] [--no-observe-pointers] [--max-instructions MAX_INSTRUCTIONS] [--timeout-ms TIMEOUT_MS] abi_contract output

Example

x86decomp harness-generate ./contract.json ./output.json

Arguments

ArgumentWhat it controls
abi_contract
required		Controls abi contract for this command.
output
required		Path where the generated output is written.
--pointer-parametersControls pointer parameters for this command.
--no-observe-pointersDisable observe pointers for this command.
--max-instructions
default: 100000		Controls max instructions for this command.
--timeout-ms
default: 1000		Controls timeout ms for this command.
