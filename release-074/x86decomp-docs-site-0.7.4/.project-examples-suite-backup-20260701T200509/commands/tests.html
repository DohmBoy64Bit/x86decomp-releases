x86decomp tests add

Adds a record to the selected regression tests.

Usage

x86decomp tests add [-h] --applicability-json APPLICABILITY_JSON --evidence-json EVIDENCE_JSON name scope_kind scope_id test_kind relative_path content_file

Example

x86decomp tests add example function pe-rva:00001000 analysis src/candidate.c ./input.json --applicability-json ./output.json --evidence-json ./evidence.json

Arguments

ArgumentWhat it controls
name
required		Controls name for this command.
scope_kind
required		Controls scope kind for this command.
scope_id
required		Identifier of the scope used by the command.
test_kind
required		Controls test kind for this command.
relative_path
required		Path to the relative used by the command.
content_file
required		Controls content file for this command.
--applicability-json
required		Path to the JSON file containing applicability data.
--evidence-json
required		Path to the JSON file containing supporting evidence.

x86decomp tests list

Example

Arguments

x86decomp tests promote-counterexample
