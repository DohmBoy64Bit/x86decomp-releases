x86decomp headers

Build and validate header ownership, declarations, and include relationships.

currentcanonical group7 runnable paths

x86decomp headers create

Creates a new header models record.

Usage

x86decomp headers create [-h] [--visibility VISIBILITY] path

Example

x86decomp headers create ./input.json

Arguments

ArgumentWhat it controls
path
required		Controls path for this command.
--visibility
default: private		Controls visibility for this command.
canonicalreconstruction

x86decomp headers cycles

Finds dependency cycles in header models.

Usage

x86decomp headers cycles [-h]

Example

x86decomp headers cycles

Arguments

This command has no additional arguments.

canonicalreconstruction

x86decomp headers declare

Adds a declaration to header models.

Usage

x86decomp headers declare [-h] [--kind KIND] [--confidence CONFIDENCE] [--evidence-json EVIDENCE_JSON] header_id symbol_id declaration

Example

x86decomp headers declare example-001 example-001 example

Arguments

ArgumentWhat it controls
header_id
required		Identifier of the header used by the command.
symbol_id
required		Identifier of the symbol used by the command.
declaration
required		Controls declaration for this command.
--kind
default: function		Category assigned to the record.
--confidence
default: 1.0		Confidence value recorded for the result.
--evidence-jsonPath to the JSON file containing supporting evidence.

x86decomp headers include

Adds an include relationship to header models.

Usage

x86decomp headers include [-h] --reason REASON source_header_id target_header_id

Example

x86decomp headers include ./candidate.c ./target.exe --reason reviewed-change

Arguments

ArgumentWhat it controls
source_header_id
required		Identifier of the source header used by the command.
target_header_id
required		Identifier of the target header used by the command.
--reason
required		Controls reason for this command.
canonicalreconstruction
