x86decomp campaign branch

Creates a controlled branch of the selected campaigns.

Usage

x86decomp campaign branch [-h] [--parent PARENT] campaign_id name

Example

x86decomp campaign branch campaign-main example

Arguments

ArgumentWhat it controls
campaign_id
required		Identifier of the campaign being operated on.
name
required		Controls name for this command.
--parentControls parent for this command.
canonicalgovernance
