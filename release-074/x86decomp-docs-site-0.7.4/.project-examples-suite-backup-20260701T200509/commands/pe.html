x86decomp pe export-coff

Exports PE reconstruction data as a portable artifact.

Usage

x86decomp pe export-coff [-h] [--section SECTION] image output

Example

x86decomp pe export-coff ./target.exe ./output.json

Arguments

ArgumentWhat it controls
image
required		Controls image for this command.
output
required		Path where the generated output is written.
--sectionControls section for this command.

x86decomp pe patch-apply

Applies an approved patch to PE reconstruction data.

Usage

x86decomp pe patch-apply [-h] plan_id

Example

x86decomp pe patch-apply example-001

Arguments

ArgumentWhat it controls
plan_id
required		Identifier of the plan used by the command.

x86decomp pe patch-plan

Creates a reviewable patch plan for PE reconstruction data.

Usage

x86decomp pe patch-plan [-h] original output operations_json

Example

x86decomp pe patch-plan ./original.bin ./output.json ./target.exe

Arguments

ArgumentWhat it controls
original
required		Controls original for this command.
output
required		Path where the generated output is written.
operations_json
required		Path to the JSON file containing operations data.
canonicalnative

x86decomp pe text-swap

Replaces the PE text section using reviewed PE reconstruction data.

Usage

x86decomp pe text-swap [-h] [--section-name SECTION_NAME] original replacement output

Example

x86decomp pe text-swap ./original.bin example ./output.json

Arguments

ArgumentWhat it controls
original
required		Controls original for this command.
replacement
required		Controls replacement for this command.
output
required		Path where the generated output is written.
--section-name
default: .text		Controls section name for this command.
