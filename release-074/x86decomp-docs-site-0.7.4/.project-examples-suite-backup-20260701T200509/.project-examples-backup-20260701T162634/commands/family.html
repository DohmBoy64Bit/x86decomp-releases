x86decomp family add

Adds a record to the selected related-function families.

Usage

x86decomp family add [-h] [--metadata-json METADATA_JSON] family_id label path

Example

x86decomp family add example-001 example ./input.json

Arguments

ArgumentWhat it controls
family_id
required		Identifier of the family used by the command.
label
required		Controls label for this command.
path
required		Controls path for this command.
--metadata-jsonPath to the JSON file containing metadata data.
canonicalgovernance

x86decomp family correlate

Correlates related evidence in related-function families.

Usage

x86decomp family correlate [-h] [--block-size BLOCK_SIZE] left_member_id right_member_id

Example

x86decomp family correlate example-001 example-001

Arguments

ArgumentWhat it controls
left_member_id
required		Identifier of the left member used by the command.
right_member_id
required		Identifier of the right member used by the command.
--block-size
default: 64		Controls block size for this command.
