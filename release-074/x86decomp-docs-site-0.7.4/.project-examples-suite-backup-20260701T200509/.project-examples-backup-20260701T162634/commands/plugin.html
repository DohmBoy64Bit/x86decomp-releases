x86decomp plugin doctor

Checks the local environment required for plugins.

Usage

x86decomp plugin doctor [-h] plugin_id

Example

x86decomp plugin doctor example-001

Arguments

ArgumentWhat it controls
plugin_id
required		Identifier of the plugin used by the command.
canonicalgovernance

x86decomp plugin install

Installs and registers plugins.

Usage

x86decomp plugin install [-h] manifest

Example

x86decomp plugin install ./output.json

Arguments

ArgumentWhat it controls
manifest
required		Path to the manifest describing the input set.
canonicalgovernance

x86decomp plugin invoke

Invokes the selected plugins.

Usage

x86decomp plugin invoke [-h] [--timeout TIMEOUT] plugin_id capability request_json

Example

x86decomp plugin invoke example-001 example ./output.json

Arguments

ArgumentWhat it controls
plugin_id
required		Identifier of the plugin used by the command.
capability
required		Controls capability for this command.
request_json
required		Path to the JSON file containing request data.
--timeout
default: 60		Maximum number of seconds allowed for the operation.
canonicalgovernance
