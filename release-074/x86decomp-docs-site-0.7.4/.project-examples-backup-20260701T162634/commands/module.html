x86decomp module add-member

Adds member to the selected project modules.

Usage

x86decomp module add-member [-h] [--ordinal ORDINAL] [--evidence-json EVIDENCE_JSON] module_id member_kind member_id

Example

x86decomp module add-member example-001 analysis example-001

Arguments

ArgumentWhat it controls
module_id
required		Identifier of the module used by the command.
member_kind
required		Controls member kind for this command.
member_id
required		Identifier of the member used by the command.
--ordinalControls ordinal for this command.
--evidence-jsonPath to the JSON file containing supporting evidence.

x86decomp module create

Creates a new project modules record.

Usage

x86decomp module create [-h] [--kind KIND] [--source-path SOURCE_PATH] [--confidence CONFIDENCE] [--evidence-json EVIDENCE_JSON] name

Example

x86decomp module create example

Arguments

ArgumentWhat it controls
name
required		Controls name for this command.
--kind
default: static-library		Category assigned to the record.
--source-pathPath to the source file used by the command.
--confidence
default: 1.0		Confidence value recorded for the result.
--evidence-jsonPath to the JSON file containing supporting evidence.
canonicalreconstruction

x86decomp module create-unit

Usage

x86decomp module create-unit [-h] [--module-id MODULE_ID] [--language LANGUAGE] [--confidence CONFIDENCE] [--evidence-json EVIDENCE_JSON] source_path
