x86decomp reloc

Inspect, resolve, and list supported relocation types.

currentcanonical group3 runnable paths

Quick example

x86decomp reloc --help
canonicalassembly

x86decomp reloc inspect

Inspects detailed relocation data.

Usage

x86decomp reloc inspect [-h] [--symbol SYMBOL] object

Example

x86decomp reloc inspect ./candidate.obj

Arguments

ArgumentWhat it controls
object
required		Controls object for this command.
--symbolSymbol name used for the operation.
canonicalassembly

x86decomp reloc resolve

Resolves the selected issue in relocation data.

Usage

x86decomp reloc resolve [-h] [--image-base IMAGE_BASE] object symbol base_rva symbol_map output

Example

x86decomp reloc resolve ./candidate.obj example 0x400000 ./target.map ./output.json

Arguments

ArgumentWhat it controls
object
required		Controls object for this command.
symbol
required		Symbol name used for the operation.
base_rva
required		Controls base rva for this command.
symbol_map
required		Path to the symbol-address mapping.
output
required		Path where the generated output is written.
--image-baseImage base used to translate relative and absolute addresses.
canonicalassembly

x86decomp reloc supported

Lists the supported forms of relocation data.

Usage

x86decomp reloc supported [-h]

Example

x86decomp reloc supported

Arguments

This command has no additional arguments.
