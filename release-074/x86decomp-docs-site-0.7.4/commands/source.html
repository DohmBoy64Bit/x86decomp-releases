x86decomp source impact

Reports the downstream impact of a change to source records.

Usage

x86decomp source impact [-h] path

Example

x86decomp source impact ./input.json

Arguments

ArgumentWhat it controls
path
required		Controls path for this command.

x86decomp source lock

Locks the selected source records against conflicting edits.

Usage

x86decomp source lock [-h] --reason REASON path

Example

x86decomp source lock ./input.json --reason reviewed-change

Arguments

ArgumentWhat it controls
path
required		Controls path for this command.

required		Controls reason for this command.

x86decomp source reconcile

Reconciles source records with current evidence.

Usage

x86decomp source reconcile [-h] [--before-sha256 BEFORE_SHA256] [--semantic {true,false}] path

Example

x86decomp source reconcile ./input.json
