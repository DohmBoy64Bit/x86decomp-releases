x86decomp proof evaluate

Usage

Example

ArgumentWhat it controls

x86decomp proof export

Exports proof records as a portable artifact.

Usage

x86decomp proof export [-h] [--include INCLUDE] output

Example

x86decomp proof export ./output.json

Arguments

ArgumentWhat it controls
output
required		Path where the generated output is written.

default: []		Controls include for this command.

x86decomp proof obligation

Creates a proof obligation for proof records.

Usage

x86decomp proof obligation [-h] [--assumptions-json ASSUMPTIONS_JSON] scope_kind scope_id property_name required_status

Example

x86decomp proof obligation function pe-rva:00001000 ./target.exe active

Arguments

ArgumentWhat it controls
scope_kind
required		Controls scope kind for this command.
scope_id
required		Identifier of the scope used by the command.
property_name
required		Controls property name for this command.
required_status
required		Controls required status for this command.
--assumptions-jsonPath to the JSON file containing assumptions data.
canonicalgovernance

x86decomp proof result

Records a proof result for proof records.

Usage

x86decomp proof result [-h] [--artifact-sha256 ARTIFACT_SHA256] obligation_id status validator report_json

Example

x86decomp proof result example-001 active example-001 ./output.json

Arguments

ArgumentWhat it controls
obligation_id
required		Identifier of the obligation used by the command.
status
required		Requested or reported status.
validator
required		Controls validator for this command.
report_json
required		Path to the JSON file containing report data.
--artifact-sha256Controls artifact sha-256 for this command.
