x86decomp library identify

Identifies the selected component using library findings.

Usage

x86decomp library identify [-h] [--version-range VERSION_RANGE] --confidence CONFIDENCE --evidence-json EVIDENCE_JSON subject_id library_name

Example

x86decomp library identify example-001 ./library.lib --confidence 1.0 --evidence-json ./evidence.json

Arguments

ArgumentWhat it controls
subject_id
required		Identifier of the project item being analyzed.
library_name
required		Controls library name for this command.
--version-rangeControls version range for this command.
--confidence
required		Confidence value recorded for the result.
--evidence-json
required		Path to the JSON file containing supporting evidence.
