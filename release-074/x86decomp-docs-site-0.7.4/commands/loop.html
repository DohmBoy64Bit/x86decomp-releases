x86decomp loop run

Runs the current workflow for closed-loop runs.

Usage

x86decomp loop run [-h] [--symbol SYMBOL] [--policy POLICY] [--timeout TIMEOUT] [--execute] function_id source compile_command_json candidate original rva slot_size

Example

x86decomp loop run pe-rva:00001000 ./candidate.c ./compile-command.json ./candidate.bin ./original.bin 0x1000 1

Arguments

ArgumentWhat it controls
function_id
required		Stable project identifier for the target function.
source
required		Path to the source input.
compile_command_json
required		Path to the JSON file containing compile command data.
candidate
required		Controls candidate for this command.
original
required		Controls original for this command.
rva
required		Relative virtual address of the target.
slot_size
required		Controls slot size for this command.
--symbolSymbol name used for the operation.
--policy
default: trailing-padding		Controls policy for this command.
--timeout
default: 120		Maximum number of seconds allowed for the operation.
--executeControls execute for this command.

x86decomp loop show

Shows one closed-loop runs record and its current state.

Usage

x86decomp loop show [-h] loop_id

Example

x86decomp loop show example-001

Arguments

ArgumentWhat it controls
loop_id
required		Identifier of the loop used by the command.
