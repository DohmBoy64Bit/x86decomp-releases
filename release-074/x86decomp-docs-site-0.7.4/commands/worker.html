x86decomp worker

Register workers, inspect capabilities, select workers, and review status.

currentcanonical group5 runnable paths

Quick example

x86decomp worker --help

x86decomp worker doctor

Checks the local environment required for workers.

Usage

x86decomp worker doctor [-h] worker_id

Example

x86decomp worker doctor worker-local

Arguments

ArgumentWhat it controls
worker_id
required		Identifier of the worker being operated on.

x86decomp worker register

Registers a new workers record.

Usage

x86decomp worker register [-h] [--endpoint ENDPOINT] [--environment-sha256 ENVIRONMENT_SHA256] name capabilities_json

Example

x86decomp worker register example ./output.json

Arguments

ArgumentWhat it controls
name
required		Controls name for this command.
capabilities_json
required		Path to the JSON file containing capabilities data.
--endpointControls endpoint for this command.
--environment-sha256Controls environment sha-256 for this command.
canonicalgovernance

Usage

Example

x86decomp worker select ./output.json

Arguments

ArgumentWhat it controls
required_json
required		Path to the JSON file containing required data.

x86decomp worker status

Shows the current status of workers.

Usage

x86decomp worker status [-h] worker_id status

Example

x86decomp worker status worker-local active

Arguments

ArgumentWhat it controls
worker_id
required		Identifier of the worker being operated on.
status
required		Requested or reported status.
