x86decomp build add-target

Adds target to the selected build models.

Usage

x86decomp build add-target [-h] [--kind KIND] [--output-name OUTPUT_NAME] [--sources-json SOURCES_JSON] [--dependencies-json DEPENDENCIES_JSON] build_id name

Example

x86decomp build add-target example-001 example

Arguments

ArgumentWhat it controls
build_id
required		Identifier of the build used by the command.
name
required		Controls name for this command.
--kind
default: executable		Category assigned to the record.
--output-nameControls output name for this command.
--sources-jsonPath to the JSON file containing sources data.
--dependencies-jsonPath to the JSON file containing dependencies data.

x86decomp build add-variant

Adds variant to the selected build models.

Usage

x86decomp build add-variant [-h] --compiler COMPILER --linker LINKER [--compile-flags-json COMPILE_FLAGS_JSON] [--link-flags-json LINK_FLAGS_JSON] [--environment-json ENVIRONMENT_JSON] target_id name

Example

x86decomp build add-variant ./target.exe example --compiler clang --linker lld-link

Arguments

ArgumentWhat it controls
target_id
required		Identifier of the target used by the command.
name
required		Controls name for this command.
--compiler
required		Controls compiler for this command.
--linker
required		Controls linker for this command.
--compile-flags-jsonPath to the JSON file containing compile flags data.
--link-flags-jsonPath to the JSON file containing link flags data.
--environment-jsonPath to the JSON file containing environment data.

Arguments

ArgumentWhat it controls
historical_build_id
required		Identifier of the historical build used by the command.
portable_build_id
required		Identifier of the portable build used by the command.

x86decomp build create

Creates a new build models record.

Usage

x86decomp build create [-h] --mode MODE [--generator GENERATOR] [--output-root OUTPUT_ROOT] [--metadata-json METADATA_JSON] name

Example

x86decomp build create example --mode release

Arguments

ArgumentWhat it controls
name
required		Controls name for this command.
--mode
required		Controls mode for this command.
--generator
default: cmake		Controls generator for this command.
--output-root
default: build		Directory where run folders, logs, and reports are created.
--metadata-jsonPath to the JSON file containing metadata data.
canonicalreconstruction
