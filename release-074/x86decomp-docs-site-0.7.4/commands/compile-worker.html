x86decomp compile-worker

Compile worker.

Usage

x86decomp compile-worker [-h] [--isolation {local_bounded,container}] [--container-image CONTAINER_IMAGE] [--cache CACHE] [--report REPORT] profile source output

Example

x86decomp compile-worker ./input.json ./candidate.c ./output.json

Arguments

ArgumentWhat it controls
profile
required		Controls profile for this command.
source
required		Path to the source input.
output
required		Path where the generated output is written.
--isolation
choices: local_bounded, container · default: local_bounded		Controls isolation for this command.
--container-imageControls container image for this command.
--cacheControls cache for this command.
--reportPath where the report is written.
