Usage

Example

x86decomp consensus explain

Explains the current state and evidence for consensus decisions.

Usage

x86decomp consensus explain [-h] subject_kind subject_id property_name

Example

x86decomp consensus explain analysis example-001 ./target.exe

Arguments

ArgumentWhat it controls
subject_kind
required		Type of project item being analyzed.
subject_id
required		Identifier of the project item being analyzed.
property_name
required		Controls property name for this command.

x86decomp consensus record

Records a new result for consensus decisions.

Usage

x86decomp consensus record [-h] --adapter ADAPTER --adapter-version ADAPTER_VERSION --evidence-id EVIDENCE_ID --group GROUP [--confidence CONFIDENCE] subject_kind subject_id property_name value_json

Example

x86decomp consensus record analysis example-001 ./target.exe ./output.json --adapter capstone --adapter-version 1.0 --evidence-id example-001 --group independent-source

Arguments

ArgumentWhat it controls
subject_kind
required		Type of project item being analyzed.
subject_id
required		Identifier of the project item being analyzed.
property_name
required		Controls property name for this command.
value_json
required		Path to the JSON file containing value data.
--adapter
required		Controls adapter for this command.
--adapter-version
required		Controls adapter version for this command.
--evidence-id
required		Identifier of the evidence record.
--group
required · default: consensus		Controls group for this command.
--confidence
default: 1.0		Confidence value recorded for the result.

x86decomp consensus resolve

Resolves the selected issue in consensus decisions.

Usage

x86decomp consensus resolve [-h] --method METHOD --rationale RATIONALE [--lock] subject_kind subject_id property_name selected_value_json

Example

x86decomp consensus resolve analysis example-001 ./target.exe ./output.json --method evidence-weighted --rationale reviewed-change

Arguments

ArgumentWhat it controls
subject_kind
required		Type of project item being analyzed.
subject_id
required		Identifier of the project item being analyzed.
property_name
required		Controls property name for this command.
selected_value_json
required		Path to the JSON file containing selected value data.
--method
required		Controls method for this command.
--rationale
required		Controls rationale for this command.
--lockControls lock for this command.
