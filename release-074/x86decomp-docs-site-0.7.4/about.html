About x86decomp and this site

x86decomp-toolkit is an evidence-governed toolkit for authorized x86 and x86-64 Windows binary analysis, reconstruction, validation, and reproducible release work.

Documentation principles

  • End-user language before implementation detail
  • A working example for every runnable command path
  • One documented entry for every current feature module and function
  • One explanation for every test in the exact inventory
  • No unfinished sections or fabricated capabilities
  • Searchable, responsive, and usable without external web assets

Source of truth

This site was generated only from the sealed x86decomp-toolkit 0.7.4 release bundle supplied for this task.

Scope

The site explains how to use the current release. Earlier release structures and source-tree update machinery are intentionally outside the active product and documentation contract.

Authorization and safety

Use the toolkit only on binaries, systems, and environments you are permitted to analyze. Isolate native execution and review generated changes before applying them.
