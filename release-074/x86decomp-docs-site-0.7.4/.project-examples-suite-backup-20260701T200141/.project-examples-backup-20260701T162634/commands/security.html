x86decomp security finding

Records a finding in security findings.

Usage

x86decomp security finding [-h] --evidence-json EVIDENCE_JSON rule_id severity subject_id summary

Example

x86decomp security finding example-001 example example-001 example --evidence-json ./evidence.json

Arguments

ArgumentWhat it controls
rule_id
required		Identifier of the rule used by the command.
severity
required		Controls severity for this command.
subject_id
required		Identifier of the project item being analyzed.
summary
required		Controls summary for this command.
--evidence-json
required		Path to the JSON file containing supporting evidence.

x86decomp security policy

Defines or updates policy for security findings.

Usage

x86decomp security policy [-h] name policy_json

Example

x86decomp security policy example conservative

Arguments

ArgumentWhat it controls
name
required		Controls name for this command.
policy_json
required		Path to the JSON file containing policy data.

ArgumentWhat it controls
