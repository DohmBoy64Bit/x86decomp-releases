x86decomp staging

Prepare candidate source, resolve staging issues, and run compile checks.

currentcanonical group5 runnable paths

Quick example

x86decomp staging --help
canonicalnative

x86decomp staging compile-check

Runs a compile check for staged reconstruction work without committing a change.

Usage

x86decomp staging compile-check [-h] [--cwd CWD] [--timeout TIMEOUT] command_json

Example

x86decomp staging compile-check ./command.json

Arguments

ArgumentWhat it controls
command_json
required		Path to the JSON file containing command data.
--cwdControls cwd for this command.
--timeout
default: 120		Maximum number of seconds allowed for the operation.
canonicalnative

x86decomp staging generate-context

Generates the context files needed for staged reconstruction work.

Usage

x86decomp staging generate-context [-h] output sources [sources ...]

Example

x86decomp staging generate-context ./output.json ./candidate.c

Arguments

ArgumentWhat it controls
output
required		Path where the generated output is written.
sources
required		Controls sources for this command.
