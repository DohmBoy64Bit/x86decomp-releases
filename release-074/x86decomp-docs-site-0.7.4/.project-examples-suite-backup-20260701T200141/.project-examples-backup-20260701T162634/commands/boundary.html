currentcanonical group5 runnable paths

x86decomp boundary audit

Audits boundary findings and reports inconsistencies.

Usage

x86decomp boundary audit [-h] [--text-end-rva TEXT_END_RVA] inventory_json

Example

x86decomp boundary audit ./output.json

Arguments

ArgumentWhat it controls
inventory_json
required		Path to the JSON file containing inventory data.
--text-end-rvaControls text end rva for this command.
canonicalnative

x86decomp boundary show

Shows one boundary findings record and its current state.

Usage

x86decomp boundary show [-h] function_id

Example

x86decomp boundary show pe-rva:00001000

Arguments

ArgumentWhat it controls
function_id
required		Stable project identifier for the target function.
