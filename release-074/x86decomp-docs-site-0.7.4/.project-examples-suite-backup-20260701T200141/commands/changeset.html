x86decomp changeset

Record, inspect, merge, rebase, verify, and apply controlled changes.

currentcanonical group10 runnable paths

Quick example

x86decomp changeset --help
canonicalreconstruction

x86decomp changeset add-operation

Adds operation to the selected controlled changesets.

Usage

x86decomp changeset add-operation [-h] changeset_id operation_json

Example

x86decomp changeset add-operation example-001 ./target.exe

Arguments

ArgumentWhat it controls
changeset_id
required		Identifier of the changeset used by the command.
operation_json
required		Path to the JSON file containing operation data.
canonicalgovernance
