Materialize assembly, add annotations, resolve relocations, and verify round trips.

x86decomp asm annotate

Adds readable annotations to assembly output while preserving the original representation.

Usage

x86decomp asm annotate [-h] --symbol SYMBOL --rva RVA [--image-base IMAGE_BASE] [--architecture {x86,x86_64}] source output

Example

x86decomp asm annotate ./candidate.c ./output.json --symbol example --rva 0x1000

Arguments

ArgumentWhat it controls
source
required		Path to the source input.
output
required		Path where the generated output is written.
--symbol
required		Symbol name used for the operation.
--rva
required		Relative virtual address of the target.
--image-baseImage base used to translate relative and absolute addresses.
--architecture
choices: x86, x86_64 · default: x86		Target instruction-set architecture.

x86decomp asm batch

Runs assembly output processing for a manifest of items and records the result.

Usage

x86decomp asm batch [-h] [--format {bytes,annotated,mnemonic}] [--assembler-command-json ASSEMBLER_COMMAND_JSON] [--image-base IMAGE_BASE] manifest output

Example

x86decomp asm batch ./output.json ./output.json

Arguments

ArgumentWhat it controls
manifest
required		Path to the manifest describing the input set.
output
required		Path where the generated output is written.
--format
choices: bytes, annotated, mnemonic · default: bytes		Controls asm format for this command.
--assembler-command-jsonPath to the JSON file containing assembler command data.
--image-baseImage base used to translate relative and absolute addresses.

x86decomp asm materialize

Writes concrete output from assembly output.

Usage

x86decomp asm materialize [-h] [--input-kind {bytes,assembly}] --symbol SYMBOL --rva RVA [--image-base IMAGE_BASE] [--architecture {x86,x86_64}] --symbol-map SYMBOL_MAP [--assembler-command-json ASSEMBLER_COMMAND_JSON] [--timeout TIMEOUT] [--report REPORT] input output_source output_object output_resolved

Example

x86decomp asm materialize ./input.json ./generated.asm ./generated.obj ./resolved.bin --symbol example --rva 0x1000 --symbol-map ./target.map

Arguments

ArgumentWhat it controls
input
required		Path to the primary input.
output_source
required		Controls output source for this command.
output_object
required		Controls output object for this command.
output_resolved
required		Controls output resolved for this command.
--input-kind
choices: bytes, assembly · default: bytes		Controls input kind for this command.
--symbol
required		Symbol name used for the operation.
--rva
required		Relative virtual address of the target.
--image-baseImage base used to translate relative and absolute addresses.
--architecture
choices: x86, x86_64 · default: x86		Target instruction-set architecture.
--symbol-map
required		Path to the symbol-address mapping.
--assembler-command-jsonPath to the JSON file containing assembler command data.
--timeout
default: 60		Maximum number of seconds allowed for the operation.

x86decomp asm verify-roundtrip

Verifies that assembly output can be rebuilt without changing the intended bytes.

Usage

x86decomp asm verify-roundtrip [-h] --symbol SYMBOL --rva RVA [--image-base IMAGE_BASE] [--architecture {x86,x86_64}] --symbol-map SYMBOL_MAP [--assembler-command-json ASSEMBLER_COMMAND_JSON] [--timeout TIMEOUT] [--report REPORT] source original output_object output_resolved

Example

x86decomp asm verify-roundtrip ./candidate.c ./original.bin ./generated.obj ./resolved.bin --symbol example --rva 0x1000 --symbol-map ./target.map
