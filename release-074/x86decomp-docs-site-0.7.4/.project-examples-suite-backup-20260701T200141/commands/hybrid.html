x86decomp hybrid compose

Combines approved inputs into hybrid reconstruction output.

Usage

x86decomp hybrid compose [-h] run_id output

Example

x86decomp hybrid compose example-001 ./output.json

Arguments

ArgumentWhat it controls
run_id
required		Identifier of the run used by the command.
output
required		Path where the generated output is written.
canonicalassembly

x86decomp hybrid generate

Generates current output for hybrid reconstruction output.

Usage

x86decomp hybrid generate [-h] [--architecture {x86,x86_64}] [--asm-format {bytes,annotated,mnemonic}] [--image-base IMAGE_BASE] [--assembler-command-json ASSEMBLER_COMMAND_JSON] [--symbol-map SYMBOL_MAP] [--overwrite] source_project output

Example

x86decomp hybrid generate ./work ./output.json

Arguments

ArgumentWhat it controls
source_project
required		Controls source project for this command.
output
required		Path where the generated output is written.
--architecture
choices: x86, x86_64 · default: x86		Target instruction-set architecture.
--asm-format
choices: bytes, annotated, mnemonic · default: bytes		Controls asm format for this command.
--image-baseImage base used to translate relative and absolute addresses.
--assembler-command-jsonPath to the JSON file containing assembler command data.
--symbol-mapPath to the symbol-address mapping.
--overwriteControls overwrite for this command.
canonicalnative

x86decomp hybrid verify

Usage

x86decomp hybrid verify [-h] composition_id

Example

x86decomp hybrid verify example-001

Arguments

ArgumentWhat it controls
composition_id
required		Identifier of the composition used by the command.
