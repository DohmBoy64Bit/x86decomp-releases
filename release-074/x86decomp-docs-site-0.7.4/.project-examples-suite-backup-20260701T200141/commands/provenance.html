x86decomp provenance binary

Records binary provenance for provenance records.

Usage

x86decomp provenance binary [-h] [--address ADDRESS] binary_id

Example

x86decomp provenance binary target-exe

Arguments

ArgumentWhat it controls
binary_id
required		Identifier of the binary used by the command.
--addressControls address for this command.

x86decomp provenance export

Usage

Example

Arguments

ArgumentWhat it controls
output
required		Path where the generated output is written.

x86decomp provenance record

Records a new result for provenance records.

Usage

x86decomp provenance record [-h] --evidence-json EVIDENCE_JSON --confidence CONFIDENCE source_path line_start line_end binary_id address_start address_end

Example

x86decomp provenance record ./candidate.c 1 1 target-exe 0x401000 0x401000 --evidence-json ./evidence.json --confidence 1.0

Arguments

ArgumentWhat it controls
source_path
required		Path to the source file used by the command.
line_start
required		Controls line start for this command.
line_end
required		Controls line end for this command.
binary_id
required		Identifier of the binary used by the command.
address_start
required		Controls address start for this command.
address_end
required		Controls address end for this command.
--evidence-json
required		Path to the JSON file containing supporting evidence.
--confidence
required		Confidence value recorded for the result.
