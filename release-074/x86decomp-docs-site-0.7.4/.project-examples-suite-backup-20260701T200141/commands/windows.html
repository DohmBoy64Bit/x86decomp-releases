x86decomp windows

Discover Windows tools, run environment checks, and build response files.

currentcanonical group3 runnable paths

Quick example

x86decomp windows --help
canonicalnative

x86decomp windows discover-ghidra

Discovers available tools for Windows tool integration.

Usage

x86decomp windows discover-ghidra [-h] [--ghidra-home GHIDRA_HOME] [--platform-name PLATFORM_NAME]

Example

x86decomp windows discover-ghidra

Arguments

ArgumentWhat it controls
--ghidra-homeControls ghidra home for this command.
--platform-nameControls platform name for this command.

x86decomp windows response-file

Builds a Windows response file for Windows tool integration.

Usage

x86decomp windows response-file [-h] output arguments_json

Example

x86decomp windows response-file ./output.json {}

Arguments

ArgumentWhat it controls
output
required		Path where the generated output is written.
arguments_json
required		Path to the JSON file containing arguments data.
