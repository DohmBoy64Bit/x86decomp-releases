x86decomp runtime launch

Launches a bounded operation for runtime validation.

Usage

x86decomp runtime launch [-h] [--argument ARGUMENT] [--timeout TIMEOUT] [--execute] image

Example

x86decomp runtime launch ./target.exe

Arguments

ArgumentWhat it controls
image
required		Controls image for this command.
--argument
default: []		Controls argument for this command.
--timeout
default: 10		Maximum number of seconds allowed for the operation.
--executeControls execute for this command.

x86decomp runtime map-crash

Maps a runtime crash back to the relevant runtime validation.

Usage

x86decomp runtime map-crash [-h] rva

Example

x86decomp runtime map-crash 0x1000

Arguments

ArgumentWhat it controls
rva
required		Relative virtual address of the target.
