Command reference

x86decomp abi

Recover, compare, export, and verify application binary interface information.

currentcanonical group6 runnable paths

Quick example

x86decomp abi --help
canonicalreconstruction

x86decomp abi compare

Compares ABI contracts and reports meaningful differences.

Usage

x86decomp abi compare [-h] left_id right_id

Example

x86decomp abi compare example-001 example-001

Arguments

ArgumentWhat it controls
left_id
required		Identifier of the first record to compare.
right_id
required		Identifier of the second record to compare.
canonicalreconstruction

x86decomp abi export

Exports ABI contracts as a portable artifact.

Usage

x86decomp abi export [-h] contract_id output

Example

x86decomp abi export ./contract.json ./output.json

Arguments

ArgumentWhat it controls
contract_id
required		Identifier or path for the ABI contract.
output
required		Path where the generated output is written.
canonicalreconstruction

x86decomp abi recover

Recovers ABI contracts from available project evidence.

Usage

x86decomp abi recover [-h] --evidence-json EVIDENCE_JSON subject_kind subject_id architecture contract_json

Example

x86decomp abi recover analysis example-001 example ./output.json --evidence-json ./evidence.json

Arguments

ArgumentWhat it controls
subject_kind
required		Type of project item being analyzed.
subject_id
required		Identifier of the project item being analyzed.
architecture
required		Target instruction-set architecture.
contract_json
required		Path where the recovered ABI contract JSON is written.
--evidence-json
required		Path to the JSON file containing supporting evidence.
canonicalreconstruction

x86decomp abi shim

Generates an ABI compatibility shim from ABI contracts.

Usage

x86decomp abi shim [-h] [--kind KIND] contract_id source_path

Example

x86decomp abi shim ./contract.json ./candidate.c

Arguments

ArgumentWhat it controls
contract_id
required		Identifier or path for the ABI contract.
source_path
required		Path to the source file used by the command.
--kind
default: wrapped		Category assigned to the record.
canonicalreconstruction
