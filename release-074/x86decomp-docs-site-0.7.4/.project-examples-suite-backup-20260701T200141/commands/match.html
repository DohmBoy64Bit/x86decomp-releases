Compare candidates, batch matching work, and report mismatches.

currentcanonical group4 runnable paths

x86decomp match batch

Runs matching results processing for a manifest of items and records the result.

Usage

x86decomp match batch [-h] [--policy {exact,trailing-padding}] [--pad-bytes-json PAD_BYTES_JSON] original candidates_json

Example

x86decomp match batch ./original.bin ./candidate.bin

Arguments

ArgumentWhat it controls
original
required		Controls original for this command.
candidates_json
required		Path to the JSON file containing candidates data.
--policy
choices: exact, trailing-padding · default: trailing-padding		Controls policy for this command.
--pad-bytes-jsonPath to the JSON file containing pad bytes data.

x86decomp match compare

Compares matching results and reports meaningful differences.

Usage

x86decomp match compare [-h] [--policy {exact,trailing-padding}] [--pad-bytes-json PAD_BYTES_JSON] [--protected-offsets-json PROTECTED_OFFSETS_JSON] original candidate

Example

x86decomp match compare ./original.bin ./candidate.bin
