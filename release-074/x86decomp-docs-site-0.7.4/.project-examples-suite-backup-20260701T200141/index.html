What the toolkit is for

x86decomp helps authorized analysts inspect Windows PE binaries, organize reconstruction work, preserve evidence, compare candidate output, materialize byte-safe assembly, and produce reproducible verification records. It separates observations from hypotheses and reports unavailable integrations explicitly.

One current interface. The active release exposes x86decomp for toolkit operations and x86decomp-test for independent verification.
