    x86decomp.integration¶

    Manifest-driven integration scenario runner.

    This module performs real target/candidate process executions with explicit host execution consent or an external isolation wrapper. It compares only declared observations and never treats finite scenarios as universal equivalence proof.

    run_integration_scenarios¶

    Execute and compare all declared integration scenarios.

    Host execution is rejected unless both the manifest selects host_explicit and the caller supplies allow_host_execution=True. external_wrapper requires a non-empty wrapper command but the toolkit cannot independently prove the wrapper is a complete sandbox; this limitation is recorded in the report.
