    x86decomp.test_bundle¶

    Safe, static-only ingestion and analysis of user-supplied test bundles.

    A test bundle is a ZIP archive containing an explicit x86decomp-test-bundle.json manifest. The default runner never executes any supplied binary or build script. It performs integrity verification and invokes the toolkit's bounded static parsers.

    This module is intentionally strict because ZIP archives and native binaries are untrusted input even when the user has authorization to analyze them.
