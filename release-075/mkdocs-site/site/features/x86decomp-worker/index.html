x86decomp.worker

Bounded command workers with explicit isolation and provenance.

The local worker provides resource and output bounds without pretending to be a security boundary. Container mode invokes Docker or Podman with a read-only root, no network, dropped capabilities, and explicit mounts.

_preexec

Return the legacy resource-limit callback for API compatibility.

The production launcher no longer passes this callback to Popen because Python code executed between fork and exec can deadlock in a process that has already initialized threads. the exec-based resource wrapper applies the same limits in a fresh Python interpreter immediately before exec.
