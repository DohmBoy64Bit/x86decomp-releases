x86decomp.angr_backend

Optional angr comparative symbolic execution backend for bounded code blobs.

The built-in symbolic engine remains the default deterministic backend. This module integrates angr when installed and performs finite path comparison over explicit input/output registers and stack arguments. Results are scoped to the selected architecture, code blobs, path/step limits, and observable registers.
