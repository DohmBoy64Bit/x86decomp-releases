x86decomp.coff_archive

Bounded Microsoft/Unix COFF archive (.lib/.a) inspection.

The archive container format is shared with ar. Microsoft libraries additionally use linker members and import-object records. This module inventories members, resolves long names, parses linker symbol indexes when structurally valid, recognizes import objects, and invokes the normal COFF parser for embedded object members.
